# Guess the number

## Description
The program challenges users to identify a hidden number over several attempts, providing them with real-time feedback to guide them toward the correct answer.

## How it works
1:The program displays a secret number and sets the maximum number of attempts allowed.

2:A while loop manages the game flow by tracking the number of attempts.

3:The user enters a number, followed by an integer (int) to convert it to a whole number.

4:The logic compares the guess to the secret number and prints a hint (higher or lower).

5:The game ends when the player finds the number or runs out of attempts.

> The project is fully functional and I am currently adding new features to improve the user experience.
